## install

    $ sudo pacman -S intellij-idea-community-edition

## shortcut

    ctrl-alt-l: 整理代码格式
    ctrl-h: 继承树
    ctrl-F12: 查看继承的成员
    ctrl-n 或 shift-shift: search everywhere

## settings

    Malformed content of <script> tag: 默认为 error, 可以调整为 warning
    live template: 代码小模板 sout = System.out.println()
